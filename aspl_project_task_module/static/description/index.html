Project Task Modules

This module adds a new field 'Project Module' in Project Tasks. This helps bifurcating tasks by project modules. User can filter/group tasks by module. This makes Kanban view clean and compact.

Screenshots

Smart Button

It shows number of modules associated to the project in project form view.

View Module

When you click the module button, it lists all the modules created in the project.

Kanban Badge

It shows module name in a task card.

Name Filter

You can also filter the tasks by module name.

Task List View

Module name is also visible in task list view.
